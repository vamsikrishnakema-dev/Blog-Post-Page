# 🌐 Blog Post Page

This is a simple **Blog Post Page project** built using pure HTML, focusing on **semantic HTML structure and accessibility**.

## Project URL

## 📌 Features

- ✅ Semantic HTML structure (`header`, `main`, `article`, `footer`)
- ✅ Proper heading hierarchy (`h1`, `h2`)
- ✅ Accessible image with `alt`, `figure`, and `figcaption`
- ✅ Blockquote with citation
- ✅ Inline and block code examples
- ✅ Descriptive links
- ✅ Clean and readable content structure
